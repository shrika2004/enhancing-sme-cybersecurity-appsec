Enhancing Small Business Cybersecurity through Application Security

This application is a lightweight security monitoring platform designed for small and medium-sized enterprises (SMEs). It helps administrators register SME web applications, run security assessments, review vulnerabilities, and monitor findings through a centralized dashboard.

The system demonstrates how application security can be integrated into small business environments using affordable tools and structured monitoring.

Core Functions
