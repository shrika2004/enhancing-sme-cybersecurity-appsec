About the Project

Small and medium-sized enterprises often use web applications without having dedicated application security teams or structured vulnerability monitoring. This project addresses that gap by providing a lightweight security monitoring platform for SME applications.

What the system does

  • Registers SME web applications as security assessment targets
  • Performs lightweight security checks
  • Stores and displays detected findings
  • Provides recommendations for improving security
  • Maintains scan history for tracking and monitoring

Security Tools Referenced

  • Bandit: Static analysis for Python applications
  • OWASP ZAP: Dynamic security testing for web applications

This project demonstrates how small businesses can strengthen application security using a centralized and affordable monitoring system.
