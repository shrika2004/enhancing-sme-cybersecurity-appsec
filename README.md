# Enhancing Small Business Cybersecurity through Application Security

This project is a Flask-based application security monitoring platform designed for small and medium-sized enterprises (SMEs). It allows administrators to register SME web applications, run lightweight security assessments, monitor vulnerabilities, review recommendations, and track scan history.

## Features
- Admin login and registration
- Register SME target applications
- Run security scans
- View vulnerability findings
- View recommendations
- Track scan history
- Risk score dashboard

## Technologies Used
- Python
- Flask
- SQLite
- HTML/CSS

## Security Concepts Referenced
- OWASP ZAP
- Bandit
- Application Security Monitoring
- Security Headers Analysis
- Vulnerability Tracking
